Orientações de uso

Nessa página são listadas todas as regionais cadastradas. Ao clicar sobre o ícone lixeira, a regional será excluída definitivamente.
Ao clicar sobre o ícone de inclusão o usuário é direcionado para a página de inclusão de Regionais. Apenas usuários com o perfil Coordenador Nacional poderão realizar inclusões de regionais.
Na lista de Regionais, ao clicar sobre o nome da regional o sistema apresentará todos os dados do cadastro, para que os dados sejam alterados se necessário.
Apenas usuários com o perfil Coordenador Nacional poderão realizar alterações de regionais.
Dica: Utilize o campo busca, localizado acima da lista para exibir na lista apenas regionais ou estados que contenham no nome parte de uma expressão, como exemplo Pará.

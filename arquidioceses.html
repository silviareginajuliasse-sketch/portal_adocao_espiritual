Orientações de uso

Nessa página são listadas todas as arquidioceses cadastradas.
Ao clicar sobre o ícone lixeira, a arquidiocese será excluída definitivamente, caso não existam paróquias ligadas a ela.
Ao clicar sobre o ícone de inclusão o usuário é direcionado para a página de inclusão de Arquidioceses. Apenas usuários com o perfil Coordenador Nacional poderão realizar inclusões de arquidioceses.
Na lista de Arquidioceses, ao clicar sobre o nome da arquidiocese o sistema apresentará todos os dados do cadastro, para que os dados sejam alterados se necessário. Apenas usuários com o perfil Coordenador Nacional poderão realizar inclusões de arquidioceses.
Dica: Utilize o campo busca, localizado acima da lista para exibir na lista apenas arquidiocioces que constenham no nome parte de uma expressão, como exemplo Vitoria

Orientações de uso

O cadastro de arquidiocese é necessário para o cadastro de novas paróquias.
Apenas colaboradores com o Perfil Coordenador Nacional tem permissão para incluir ou alterar cadastros de Arquidioceses.
Um registro de arquidiocese não pode ser excluído se houverem Paróquias registradas na Arquidiocese. Caso deseje que o nome da Arquidiocese não seja mais apresentado no cadastro de Paróquias altere o campo Status para Inativo.
Dica: Para excluir definitivamente uma Arquidiocese, lembrando que só é possível se não houver Paróquia vinculada, clique no botão Excluir.

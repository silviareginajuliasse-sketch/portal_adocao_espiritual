Orientações de uso

Nessa página são listados todos os treinamentos cadastrados no sistema.
Ao clicar sobre o ícone lixeira, o treinamento será excluído definitivamente, caso não existam colaboradores vinculados a ele.
Ao clicar sobre o ícone de inclusão o usuário é direcionado para a página de inclusão de Treinamentos. Apenas usuários com o perfil Coordenador Nacional poderão realizar inclusões.
Na lista de Treinamentos, ao clicar sobre o nome do treinamento o sistema apresentará todos os dados do cadastro, para que sejam alterados se necessário.
Dica: Utilize o campo busca, localizado acima da lista, para filtrar os treinamentos pelo nome ou parte de uma expressão.
